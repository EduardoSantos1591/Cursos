Uma lista não ordenada começa com a tag. Cada item da lista começa com a tag.
  • Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão: Exemplo
    • Coffee
    • Tea
    • Milk
    Lista HTML não ordenada - Selecionar marcador de item da lista A list-style-typepropriedade CSS é usada para definir o estilo do marcador do item de lista: Value Description disc Sets the list item marker to a bullet (default) circle Sets the list item marker to a circle square Sets the list item marker to a square none The list items will not be marked Exemplo - Disco

    Lista HTML ordenada Uma lista ordenada começa com a tag. Cada item da lista começa com a tag.
    1. Os itens da lista serão marcados com números por padrão: Exemplo
      1. Coffee
      2. Tea
      3. Milk
      Lista HTML ordenada - o atributo Type O typeatributo da tag define o tipo do marcador do item da lista:
        Type Description type="1" The list items will be numbered with numbers (default) type="A" The list items will be numbered with uppercase letters type="a" The list items will be numbered with lowercase letters type="I" The list items will be numbered with uppercase roman numbers type="i" The list items will be numbered with lowercase roman numbers Números:
        1. Coffee
        2. Tea
        3. Milk
        Letras maiúsculas:
        1. Coffee
        2. Tea
        3. Milk
        Letras minúsculas:
        1. Coffee
        2. Tea
        3. Milk
        Números romanos maiúsculos:
        1. Coffee
        2. Tea
        3. Milk
        Números romanos em minúsculas:
        1. Coffee
        2. Tea
        3. Milk
        Listas de descrição HTML O HTML também suporta listas de descrição. Uma lista de descrição é uma lista de termos, com uma descrição de cada termo. A
        tag define a lista de descrição, a tag define o termo (nome) e a tag descreve cada termo:
        Exemplo
        Coffee
        - black hot drink
        Milk
        - white cold drink
        Listas HTML aninhadas A lista pode ser aninhada (listas dentro de listas): Exemplo
        • Coffee
        • Tea
          • Black tea
          • Green tea
        • Milk
        Contagem da lista de controle Por padrão, uma lista ordenada começará a contar a partir de 1. Se você quiser começar a contar a partir de um número especificado, poderá usar o startatributo: Exemplo
        1. Coffee
        2. Tea
        3. Milk
        Lista Horizontal com CSS As listas HTML podem ser estilizadas de várias maneiras diferentes com CSS. Uma maneira popular é estilizar uma lista horizontalmente, para criar um menu de navegação: Exemplo
        • Home
        • News
        • Contact
        • About
