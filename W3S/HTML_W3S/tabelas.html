Definindo uma tabela HTML Uma tabela HTML é definida com a tag Cada linha da tabela é definida com a tag Um cabeçalho da tabela é definido com a
tag Por padrão, os títulos das tabelas estão em negrito e centralizados. Os dados / célula da tabela são definidos com a tag Exemplo:
Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
Tabela HTML - Bordas recolhidas Se você deseja que as bordas colapsem em uma borda, adicione a border-collapse propriedade CSS : Exemplo: table, th, td { border: 1px solid black; border-collapse: collapse; } Tabela HTML - Adicionando preenchimento de célula O preenchimento de célula especifica o espaço entre o conteúdo da célula e suas bordas. Se você não especificar um preenchimento, as células da tabela serão exibidas sem preenchimento. Para definir o preenchimento, use a padding propriedade CSS : Exemplo th, td { padding: 15px; } Tabela HTML - Cabeçalhos de alinhamento à esquerda Por padrão, os títulos das tabelas estão em negrito e centralizados. Para alinhar à esquerda os títulos da tabela, use a text-alignpropriedade CSS : Exemplo th { text-align: left; } Tabela HTML - Adicionando espaçamento de borda O espaçamento da borda especifica o espaço entre as células. Para definir o espaçamento da borda de uma tabela, use a border-spacingpropriedade CSS : Exemplo table { border-spacing: 5px; } Tabela HTML - Células que abrangem muitas colunas Para fazer uma célula abranger mais de uma coluna, use o colspan atributo: Exemplo
Name Telephone
Bill Gates 55577854 55577855
Tabela HTML - Células que abrangem muitas linhas Para fazer uma célula abranger mais de uma linha, use o rowspanatributo: Exemplo
Name: Bill Gates
Telephone: 55577854
55577855
Tabela HTML - Adicionando uma legenda Para adicionar uma legenda a uma tabela, use a
tag: Exemplo
Monthly savings
Month Savings
January $100
February $50
Um estilo especial para uma mesa Para definir um estilo especial para uma tabela especial, adicione um id atributo à tabela: Exemplo
Firstname Lastname Age
Eve Jackson 94
Agora você pode definir um estilo especial para esta tabela: table#t01 { width: 100%; background-color: #f1f1c1; } E adicione mais estilos: table#t01 tr:nth-child(even) { background-color: #eee; } table#t01 tr:nth-child(odd) { background-color: #fff; } table#t01 th { color: white; background-color: black; } Resumo do capítulo Use o elemento HTML para definir uma tabela Use o elemento HTML para definir uma linha da tabela Use o
