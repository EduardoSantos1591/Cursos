Links HTML - Crie um marcador Marcadores HTML são usados ​​para permitir que os leitores pulem para partes específicas de uma página da Web. Os marcadores podem ser úteis se uma página da web for muito longa. Para criar um marcador - primeiro crie o marcador e adicione um link a ele. Quando o link é clicado, a página rola para baixo ou para cima até o local com o marcador. Exemplo Primeiro, crie um marcador com o id atributo:
