Mapas de imagem A tag define um mapa de imagem. Um mapa de imagem é uma imagem com áreas clicáveis. Exemplo Como funciona? A ideia por trás de um mapa de imagem é que você possa executar ações diferentes, dependendo da localização da imagem em que clicar. Para criar um mapa de imagem, você precisa de uma imagem e de um mapa contendo algumas regras que descrevam as áreas clicáveis. A IMAGEM A imagem é inserida usando a tag. A única diferença de outras imagens é que você deve adicionar um usemap atributo: O usemap valor começa com uma marca de hash # seguida pelo nome do mapa da imagem e é usado para criar um relacionamento entre a imagem e o mapa da imagem. O MAPA Em seguida, adicione um elemento. O elemento é usado para criar um mapa de imagem e é vinculado à imagem usando o name atributo: O name atributo deve ter o mesmo valor que o usemap atributo. As ÁREAS Em seguida, adicione as áreas clicáveis. Uma área clicável é definida usando um elemento. Forma Você deve definir a forma da área e escolher um destes valores: rect - define uma região retangular circle - define uma região circular poly - define uma região poligonal default - define toda a região COORDENADAS Você deve definir algumas coordenadas para poder colocar a área clicável na imagem. As coordenadas vêm em pares, uma para o eixo x e outra para o eixo y. As coordenadas 34, 44 estão localizadas 34 pixels da margem esquerda e 44 pixels da parte superior: As coordenadas 270, 350 estão localizadas 270 pixels da margem esquerda e 350 pixels da parte superior: Agora você tem dados suficientes para criar uma área retangular clicável: Círculo Para adicionar uma área do círculo, localize primeiro as coordenadas do centro do círculo: 337, 300 Em seguida, especifique o raio do círculo: 44 píxeis Agora você tem dados suficientes para criar uma área circular clicável: Mapa de imagem e JavaScript Uma área clicável não precisa ser um link para outra página; também pode disparar uma função JavaScript.
