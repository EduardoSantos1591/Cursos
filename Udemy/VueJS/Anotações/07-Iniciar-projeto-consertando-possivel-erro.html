Iniciando o projeto Para iniciar o projeto vá onde seu arquivo db.json foi criado, no meu caso foi "introducao_vue_js" e execute o comando "json-server --watch db.json". Após isso vá a pasta raiz do seu projeto, no caso é a netflixVue e execute "npm run dev", isso depende do nome que você deu para o script dentro de package.json. Possivel Erro Após terminar o curso, eu parei de mexer no projeto, depois de um tempo por algum motivo quando tentei inicializar o projeto apareceu um erro. No próprio erro me mostra uma possivel solução: "npm rebuild node-sass" após isso basta iniciar o projeto novamente e já vai funcionar.
