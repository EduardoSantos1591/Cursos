ES é uma sigla para ECMA Script, o 6 é a sua versão. O JS é uma linguagem que esta muito em alta, por isso ela recebe atualizações e novas regras muitas vezes no ano, por isso o ECMA Script, a empresa por tras dessas regras, adicionam novas regras todo ano, elas são adicionadas por meio desses pacotes chamados de ES6 , ES7, ES8... ---------------------------------------------------------------------------------------------------------------------------------------------------------------- O que é Babel ? Como o JS se expande muito rapido os navegadores são um pouco lentos para seguir as novas diretrizes, como é o caso do Internet Explorer é um pouco mais lento com isso ele possa não entender todas as funcionalidades do ES6. O Babel basicamente pega o código que você escreve com as novas funcionalidades e converte em um código que todos os navegadores entendam. ---------------------------------------------------------------------------------------------------------------------------------------------------------------- Funcionalidades que serão mostradas no curso : Webpack - É um servidor que nos permite ter um live reload, faz com que o navegador atualize automaticamente quando escrevemos um novo código, sempre que um arquivo JS feito com ES6 for alterado automaticamente ele aciona o babel para o navegador entenda. Classes - Arrow Functions - Novo modelo para se criar funções anonimas. Desestruturação - É uma forma de recuperar propriedades de um objeto, um array de forma simples. Rest/spread - Formas de manipular as propriedades e informações dentro de objetos e arrays, duplicar esse conteúdo, repassar de um lugar para o outro mantendo a referência. Import/Export - Forma de importar e Exportar funcionalidades de um arquivo de JS para outro Async/Await - Fornece uma API nova para se trabalhar de forma assíncrona dentro do JS.
